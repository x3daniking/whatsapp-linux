Installation

Ubuntu / Pop!_OS / Debian

# Download the latest release
wget https://github.com/x3daniking/whatsapp-linux/releases/latest/download/WhatsApp-Linux-1.0.0-amd64.deb

# Install
sudo dpkg -i WhatsApp-Linux-1.0.0-amd64.deb
sudo apt-get install -f  # Fix any dependency issues

Fedora / CentOS / RHEL

# Download and install
wget https://github.com/x3daniking/whatsapp-linux/releases/latest/download/WhatsApp-Linux-1.0.0-x86_64.rpm
sudo rpm -i WhatsApp-Linux-1.0.0-x86_64.rpm

AppImage (Universal)

# Download the AppImage
wget https://github.com/x3daniking/whatsapp-linux/releases/latest/download/WhatsApp-Linux-1.0.0-x86_64.AppImage

# Make executable and run
chmod +x WhatsApp-Linux-1.0.0-x86_64.AppImage
./WhatsApp-Linux-1.0.0-x86_64.AppImage

Arch Linux

# Using AUR helper (yay)
yay -S whatsapp-linux-bin

# Manual installation
git clone https://aur.archlinux.org/whatsapp-linux-bin.git
cd whatsapp-linux-bin
makepkg -si

Building from Source

1

Prerequisites

# Ubuntu/Pop!_OS/Debian
sudo apt update
sudo apt install nodejs npm git build-essential

# Additional build dependencies
sudo apt install libnss3-dev libatk-bridge2.0-dev libdrm2-dev libxcomposite-dev libxdamage-dev libxrandr-dev libgbm-dev libxss-dev libasound2-dev
2

Clone and Build

# Clone the repository
git clone https://github.com/x3daniking/whatsapp-linux.git
cd whatsapp-linux

# Install dependencies
npm install

# Build TypeScript
npm run build
3

Create Packages

# Build specific packages
./build.sh deb      # For DEB package
./build.sh appimage # For AppImage
./build.sh all      # For all packages

# Or use make
make deb            # Build DEB package
make appimage       # Build AppImage
make all            # Build all packages

System Requirements

Minimum

  • OS: Ubuntu 18.04+, Debian 10+
  • RAM: 512 MB
  • Storage: 200 MB free space
  • Architecture: x86_64 (64-bit)

Recommended

  • RAM: 1 GB or more
  • Network: Stable internet connection
  • Display: 1024x768 or higher
  • Audio: PulseAudio support
